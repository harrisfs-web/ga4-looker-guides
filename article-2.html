Step 2: Essential Metrics Selection

Metric Type Examples Use Case
User Engagement Active Users, Session Duration Understanding user behavior
Acquisition Traffic Source, Campaign Marketing effectiveness
Conversion Conversion Rate, Revenue Business performance

Custom Metric Formula Example:

// Revenue per User
SUM(revenue) / COUNT(DISTINCT user_id)

// Engagement Rate
engaged_sessions / total_sessions * 100

// Custom Score
CASE
  WHEN session_duration > 180 THEN 3
  WHEN session_duration > 60 THEN 2
  ELSE 1
END
